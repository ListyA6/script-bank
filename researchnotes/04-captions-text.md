# Captions & On-Screen Text — Operator/Builder Shorts

## 1. Caption styles & when each fits

- **Word-by-word karaoke** (one word pops at a time): highest measured retention on opinion/rant content. Hormozi, motivational-edge. Use sparingly for operator content — feels salesy if overused. Good for hype beats inside a longer talking head.
- **Sentence-chunked / 3–5 word phrases**: the default for builder content (tutorials, breakdowns, "here's what I shipped"). Each clause gets a moment to land. Theo, Greg Isenberg, Sam Parr-style podcast clips.
- **Hook-pinned** (sticky line top + dynamic captions bottom): the strongest single combo. Top stays for the first 3–7s (sometimes the whole clip), bottom updates. Standard on Submagic/Opus templates.
- **Hybrid karaoke + chunk**: 2–4 words appear together, current word highlights. Lowest cognitive load, most "modern operator" feel. This is Submagic's default "Hormozi-evolved" preset.

Builder rule: if the script is dense (numbers, frameworks, code), go chunked. If it's a narrative or punchline, go karaoke.

## 2. Fonts — builder credible vs cheesy

- **Builder credible**: Inter, Geist, SF Pro, Helvetica Neue, Söhne, IBM Plex Sans, JetBrains Mono (for code/number flexes). Theo, Ben Lang, Vassallo lean clean sans, often medium weight, tight tracking, white-on-dark with thin stroke or shadow.
- **High-contrast operator (Hormozi lane)**: Montserrat Black / Anton / Bebas Neue, ALL CAPS, thick black stroke, yellow keyword highlight. Reads as "internet entrepreneur." Works for Hormozi because his persona is overtly sales — for a quieter operator brand it tips cheesy fast.
- **Cheesy / avoid**: Komika Axis, Impact with rainbow gradient, TikTok default "Brat" preset, anything that ships as the platform default. MrBeast bright-red drop-shadow Impact = entertainer signal, not builder.

Pick one font, lock weight + size + stroke, reuse across every clip. Consistency *is* the brand.

## 3. Highlight / keyword treatment

- **Rule of thumb**: 1–2 highlighted words per phrase, never more. Highlight nouns, numbers, proper names, verbs of action — never articles or filler.
- **Treatments**: color swap (yellow/green/brand accent), scale pop (110–120% for 100ms), or box-fill background. Pick *one* treatment per video.
- **Builder palette**: single accent color matching brand (Theo orange, Levels neon green, Hormozi yellow). Avoid >2 highlight colors in one clip.
- **Animation**: pop-in under 120ms. Slower = MrBeast-feel. No bouncing, no rotation.

## 4. Pinned / sticky text

The persistent top caption is the hook restated. Operator pattern: 3–6 word promise, sentence case or all-caps, sits in top 15% with a subtle dark plate so it survives any background. Examples: "How I run 3 restaurants from my laptop." "What $0 → $10K MRR actually looks like." Pin for the first 5–10s, or full clip if <30s. Submagic, Opus and CapCut all ship this as "title" or "headline" layer.

## 5. Numbers, currency, metrics

This is where builder content earns trust. Treat numbers as their own design element:

- Show in monospace or tabular-figure font so digits don't jitter.
- Animate with a count-up (After Effects slider, ShowNumbers, Krumzi) on the *first* reveal of a metric. Don't count up every time.
- Pair the number with a unit chip ("$10K **MRR**", "3 **restaurants**", "130M **IDR/mo**") — the unit smaller and muted, the number large and accented.
- Hold the number on screen 1.5–2s minimum so it can be screenshotted. Screenshots = distribution.

## 6. Title cards / chapter markers in <60s

For sub-60s shorts: skip true chapter cards. Instead use a 0.4s full-frame text flash ("STEP 1", "THE PROBLEM", "RESULT") with a hard cut sound. Acts as a chapter without burning runtime. For 60–90s, a thin progress bar or "1/3" counter top-right works. Builders use sparingly — Ben Lang, Tibo — to signal "this is structured, not rambling."

## 7. Subtitles vs captions vs lower-thirds

- **Subtitles** = translation of dialogue, assumed audience doesn't speak source language. SRT track.
- **Captions** = transcription + non-speech cues, burned-in for sound-off viewing. This is what 80% of viral shorts use.
- **Lower-thirds** = name/title/handle plate, identifies speaker or source. Operator use: pin your @ + role ("@listy / 3 restaurants, 1 agency") for first 3s of every clip. Build recognition.

Don't conflate. A short can carry all three.

## 8. Bilingual (Indonesian → English-first)

International builder audience = English captions, always. Options:

- **English-only burned-in** + Indonesian SRT track uploaded to YouTube/IG: cleanest, lets algorithm serve translation. Default choice.
- **Dual-stack** (EN top, ID smaller below): only for content explicitly aimed at both markets. Eats frame, looks busy. Skip for personal brand.
- If speaking Indonesian on camera, burn English captions and let voice be texture. Levels does this in reverse (Dutch accent, English captions clean it up).

Never machine-translate idioms without review — "warung", "nasi", brand names stay untranslated, italicized.

## 9. Automated tools — builder-grade vs template-screams

- **Submagic**: best engine in 2026, 99%+ accuracy, but the default templates scream "I bought Submagic." Buy it, then strip the preset down — own font, one accent color, no emojis, no auto-zoom. Builder-grade after customization.
- **Captions.ai**: most "viral" presets, leans Hormozi-evolved with 3D and emojis. Hard to make look minimal. Skip for builder tone.
- **CapCut auto-captions**: free, decent accuracy, fully customizable. With a saved style preset, indistinguishable from Submagic output. Best price/quality for operators doing volume.
- **Opus Clip**: optimizes for finding clips, not finishing them. Captions are generic. Use for sourcing, finish elsewhere.
- **Manual in Premiere/DaVinci**: only worth it for hero pieces. Auto-transcribe → restyle in a saved title preset.

Operator default stack: CapCut auto-captions with a locked custom preset (Inter Bold, white, 4px black stroke, one accent color for highlights, top-pinned hook plate). Reusable, free, no template signature.
