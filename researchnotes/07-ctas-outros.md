# CTAs & Outros for Operator/Builder Short-Form

## 1. CTA Taxonomy (operator/builder flavor)

| CTA | Use when | Example |
|---|---|---|
| Follow for X | First-time viewer, you have a clear niche promise | "I post one of these every time I ship. Follow if you build things." |
| Comment "Y" for the doc | You actually have a lead magnet (notion doc, repo, template) | "Comment 'STACK' and I'll DM the exact tools." |
| Link in bio | You have ONE link, and the video literally previews what's behind it | Marc Lou style: video shows the dashboard, bio = ShipFast. |
| "I'll DM you" | Niche, high-intent, low volume — you can actually DM 50 people | Ben Lang-style community plays |
| "Part 2 tomorrow" | You actually filmed part 2. Otherwise it burns trust | Series breakdowns of a build |
| "Save this" | Reference content (checklist, framework, prices) | Greg Isenberg list-style reels |
| No CTA | The video is the product (build log, demo, screen-record) | Pieter Levels demoing a metric on his dashboard |

## 2. When NO CTA wins
Pieter Levels, Theo, and Marc Lou frequently end on the artifact itself — a revenue chart, a deployed feature, a `git push`. The loop is: see the thing → click profile → see more things → follow. Adding "follow for more" on top of a strong build clip dilutes the proof. Content-led growth assumes the work does the asking. Rule: if the last frame already provokes "who is this guy," shut up.

## 3. Soft vs Hard CTAs
Hard (avoid): "FOLLOW NOW for more dev tips!" — guru-coded, hits the same nerve as the YouTube bell.
Soft (default): conditional + identity-based.
- "If you ship things on weekends, this account is for you."
- "Following along as I take this from 0 to $1k MRR."
- "More of this if you're into small bets."
Vassallo's rule: one CTA, end of piece, framed as an invitation not a demand. Sam Parr leans on curiosity tails ("the rest is in Friday's email") instead of imperatives.

## 4. Ask Only After Value
Placement: CTA goes AFTER the payoff, never before or during. Standard short-form structure: Hook (0-3s) → Value/Story (4-40s) → Payoff (last beat) → CTA (final 2-3s, optional). If the viewer hasn't received the thing you promised in the hook, no CTA will convert. Specific CTAs outperform generic ones ~3:1 (HubSpot 2025), but only when earned.

## 5. Outro Formats — Tradeoffs

| Format | How it works | Best for | Risk |
|---|---|---|---|
| Loop-back | Last line sets up the first line; video replays seamlessly | Watch-time hacking, algorithmic boost | Feels manipulative if loop is obvious |
| Payoff drop | End on the artifact (chart, ship, before/after) | Build-in-public, demos | Wastes opportunity if profile isn't dialed |
| Single-frame card | Last 1.5s = static frame, handle + one line | Brand recall, cross-platform | Boring if overused |
| Talking-head sign-off | "That's it. I'm Listy. I build restaurants and software." | Personality-driven, niche claim | Cringe if forced; needs natural delivery |

Tibo and Sam Parr lean on payoff-drop + curiosity tail. Greg Isenberg's faceless formula uses single-frame card. Levels rarely signs off — he just stops.

## 6. Funnel Off-Platform
Operator/builder funnel order (least to most friction):
1. Profile follow (in-app)
2. Bio link → one destination (newsletter OR product, never both)
3. "Comment X → DM" → email capture in DM
4. Newsletter → product

Marc Lou funnels short-form → Twitter → ShipFast. Sam Parr funneled everything → The Hustle email. Rule: pick ONE off-platform destination per quarter. Don't send the same video's viewers to a newsletter AND a Gumroad AND a Discord.

## 7. Indonesia / SEA Adaptation
- "Comment X" works, but warmer framing converts better. Use "teman-teman" register, not transactional "klik sekarang."
- Community-belonging CTAs ("gabung" / "join") outperform urgency ("terakhir hari ini").
- Indonesian audiences over-index on comments vs likes — a soft "kasih tau di komen…" reliably triggers replies where English audiences would scroll.
- For an operator-builder positioning aimed at international audiences from Indonesia: keep English CTAs simple and identity-based; don't code-switch mid-CTA, it breaks the frame.
- DM-based CTAs work especially well in SEA (WhatsApp culture transfers to IG DMs).

## 8. Anti-Patterns (do not do)
- "Don't forget to like, comment, and subscribe" — YouTube 2014 energy
- "Follow for part 2!!!" when part 2 doesn't exist
- "Only 24 hours left" fake urgency on evergreen content
- Stacking 3 CTAs (follow + comment + link) in one outro — picks zero
- Reading the CTA in a different, louder tone than the rest of the video
- "Wait for it…" with no payoff
- Begging ("we're so close to 10k, please follow")
- Hand-on-screen "tap follow" gestures — guru-coded
- CTA before value is delivered
- Generic "more tips coming" with no niche specificity
